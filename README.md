# 正在学刷课助手 ⚡

> Chrome 浏览器插件，适用于 [learnin.com.cn（正在学）](https://www.learnin.com.cn) 平台
>
> 当前版本：**v1.3.0** · [查看更新日志](./CHANGELOG.md)

## ✨ 功能

| 功能 | 说明 |
|------|------|
| 🎬 自动播放 | 视频页加载后自动开始播放，无需手动点击 |
| ⚡ 倍速播放 | 支持 1× / 1.5× / 2× / 3×，默认 3× |
| ⏭ 自动切换下一节 | 视频播完自动跳转下一课时（默认关闭） |
| 🚫 自动关闭弹窗 | 自动点掉"请认真观看"等提示 |
| 🗂 一键多开 | 在课程列表页一键将所有视频分别在新标签页打开，并行自动播放 |
| 🖱 悬浮控制面板 | 页面右下角，可拖拽移动，可收起 |

## 🚀 安装方法

1. 下载本仓库（点击右上角 `Code → Download ZIP`，解压）
2. 打开 Chrome，地址栏输入 `chrome://extensions/`
3. 右上角开启 **「开发者模式」**
4. 点击 **「加载已解压的扩展程序」**
5. 选择解压后的文件夹
6. 安装完成 ✅

## 📖 使用方法

### 普通刷课

1. 登录 [learnin.com.cn](https://www.learnin.com.cn) 并进入课程视频页面
2. 页面右下角出现 **⚡ 悬浮控制面板**
3. 选择倍速，开启需要的功能
4. 点击 **「▶ 开始刷课」** 即可

### 一键多开（并行刷课）

1. 进入课程的**视频列表页**（能看到所有课时列表的页面）
2. 控制面板底部出现 **「🗂 一键多开 (N)」** 按钮，N 为检测到的视频数量
3. 点击后每隔 600ms 自动在新标签页打开一个视频
4. 每个标签页独立运行插件，**自动播放 + 3× 倍速**，无需任何操作

> ⚠️ **提示**：首次多开时浏览器可能提示"已拦截弹出式窗口"，需在地址栏右侧点击**允许 learnin.com.cn 的弹出式窗口**后重试。
