# 正在学刷课助手 ⚡

> Chrome 浏览器插件，适用于 [learnin.com.cn（正在学）](https://www.learnin.com.cn) 平台

## ✨ 功能

| 功能 | 说明 |
|------|------|
| 🎬 自动播放 | 视频加载后自动开始播放 |
| ⚡ 倍速播放 | 支持 1× / 1.5× / 2× / 3× |
| ⏭ 自动切换下一节 | 视频播完自动跳转下一课时 |
| 🚫 自动关闭弹窗 | 自动点掉"请认真观看"等提示 |
| 🖱 悬浮控制面板 | 页面右下角，可拖拽，可收起 |

## 🚀 安装方法

1. 下载本仓库（点击右上角 `Code → Download ZIP`，解压）
2. 打开 Chrome，地址栏输入 `chrome://extensions/`
3. 右上角开启 **「开发者模式」**
4. 点击 **「加载已解压的扩展程序」**
5. 选择解压后的文件夹
6. 安装完成 ✅

## 📖 使用方法

1. 登录 [learnin.com.cn](https://www.learnin.com.cn) 并进入课程视频页面
2. 页面右下角会出现 **⚡ 悬浮控制面板**
3. 选择倍速，开启需要的功能
4. 点击 **「开始刷课」** 即可

## 📁 文件结构

```
learnin-helper/
├── manifest.json     # 插件配置 (Manifest V3)
├── content.js        # 核心刷课逻辑
├── panel.css         # 悬浮面板样式
├── popup.html        # 点击图标弹出页
├── popup.js          # 弹出页逻辑
└── icons/            # 插件图标
```
