# Changelog

所有版本变更记录。

---

## [1.2.0] - 2026-03-17

### ✨ 新增

- **一键多开**：在课程列表页（`.video-items`）自动识别所有 `a.student-course-video-item` 链接，每隔 600ms 依次在新标签页打开，每个标签页独立自动播放，实现并行刷课。
- 多开按钮智能显示：仅在检测到课程列表时显示「🗂 一键多开」按钮，并实时显示可打开的视频数量；视频播放页自动隐藏该按钮。
- SPA 路由切换时自动刷新多开按钮状态。

---

## [1.1.0] - 2026-03-17

### 🐛 Bug 修复

- **修复自动播放未触发**：平台使用 video.js 播放器，直接调用原生 `video.play()` 会被拦截。现改为优先通过 video.js 的 `player.play()` API 触发；备用方案为先静音播放（绕过浏览器自动播放策略限制），播放成功后 300ms 自动恢复音量。
- **修复倍速被平台重置**：新增对 video.js `play` / `ratechange` 事件的监听，与原生事件双重绑定，防止平台在视频切换或加载时重置播放速度。

### ✨ 新增

- 自动识别并获取 video.js 播放器实例，兼容 `video.player`、`videojs.getPlayer()` 及父容器 ID 三种方式。

## [1.0.0] - 2026-03-17

### 🎉 首次发布

- 悬浮控制面板（可拖拽、可收起）
- 倍速播放：支持 1× / 1.5× / 2× / 3×，默认 3×
- 自动播放视频
- 自动切换下一节（默认关闭）
- 自动关闭弹窗（"请认真观看"等提示）
- SPA 路由变化感知，切换课程后自动重新激活
- 支持 learnin.com.cn 全站课程页面


### 🐛 Bug 修复

- **修复自动播放未触发**：平台使用 video.js 播放器，直接调用原生 `video.play()` 会被拦截。现改为优先通过 video.js 的 `player.play()` API 触发；备用方案为先静音播放（绕过浏览器自动播放策略限制），播放成功后 300ms 自动恢复音量。
- **修复倍速被平台重置**：新增对 video.js `play` / `ratechange` 事件的监听，与原生事件双重绑定，防止平台在视频切换或加载时重置播放速度。

### ✨ 新增

- 自动识别并获取 video.js 播放器实例，兼容 `video.player`、`videojs.getPlayer()` 及父容器 ID 三种方式。
